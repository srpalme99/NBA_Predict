v7 model pipeline

Six stages turn raw NBA data into calibrated win probabilities every night.

01
Game discovery
ScoreboardV3 fetches tonight's matchups at runtime.
02
Injury scrape
ESPN endpoint parsed — Out / Doubtful / Questionable flagged per team.
03
Advanced stats fetch
LeagueDashTeamStats called with runtime kwarg probe for nba_api version compat.
04
Rolling stat calc
Last 10 games averaged for core box-score stats + 5/20-game EWMA windows.
05
Feature assembly
52 HOME − AWAY diff features. PIS blended 40% season / 60% recent form.
06
Prediction
Meta-stacker blends margin model + classifier → final win probability + confidence label.
About
Built on real data.

CourtIQ is a Python NBA prediction project using nba_api, scikit-learn, and ESPN injury data. The model runs nightly via GitHub Actions and publishes fresh picks to this page automatically.
